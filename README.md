#cse python project

The Spam Message Detector is a Python-based project used to classify messages as Spam or Not Spam (Ham).

With the growth of digital communication (emails, messaging apps, social media), spam messages have become very common.

Spam messages may include advertisements, fake offers, or phishing links, making detection important.


🔹 Approaches Used:

Keyword-based method:

Detects spam using words like “free”, “win”, “offer”.

Simple and easy to implement.


Count-based method:

Counts number of spam-related words.

Improves accuracy compared to basic method.


Function-based implementation:

Organizes code into reusable functions.

Improves readability and structure.


Machine Learning method:

Uses Naive Bayes algorithm.

Uses Count Vectorizer to convert text into numerical form.

More accurate and efficient than rule-based methods.



🔹 Working Process:

User enters a message.

Text is preprocessed (lowercase conversion, removing punctuation).

Message is converted into numerical features.

Selected method/model classifies the message.

Output is displayed as Spam or Not Spam.
